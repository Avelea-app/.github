## About

Avelea is a mobile health app designed to make menstrual cycle tracking feel simple, supportive, and visually gentle.

The project is being built with a strong focus on clarity, emotional comfort, and clean user experience. The goal is to create a product that feels calm, trustworthy, and easy to use in everyday life.

## Focus

- menstrual cycle tracking
- supportive and calm UX
- clean visual design
- mobile-first product development

## Tech

- Flutter
- Dart
- Android / iOS
- GitHub for development workflow

## Status

Avelea is currently in development.  
The foundation of the application is being built step by step, including project structure, theming, and core architecture.
