## About

Avelea is a mobile health app created to make menstrual cycle tracking feel simple, supportive, and visually gentle.

The project focuses on clarity, emotional comfort, and clean user experience, with a calm interface and thoughtful structure at its core.

## Focus

- menstrual cycle tracking
- supportive and calm UX
- clean visual design
- mobile-first development
